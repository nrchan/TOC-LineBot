# TOC Project 2020

## 介紹

### 簡單來說...
這個 bot 能~~幫我~~幫使用者查詢各個和弦和音階的組成音，也可以進行反向查詢。
### 功能
一開始會出現選單。可以查詢「和弦名稱」、「和弦組成音」、「音階名稱」還有「音階組成音」。 
> 可以直接點選需要的功能，或是自行輸入。
